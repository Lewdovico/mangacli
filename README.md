# mangacli
A bash script to view manga in your command line.

I created this script because I wanted to view my favorite manga on sxiv.

## Requirements

1. curl
2. jq
3. sed
4. cut
5. pup
6. cat
7. grep
8. awk

## Usage

```
mangacli <search term>
```
